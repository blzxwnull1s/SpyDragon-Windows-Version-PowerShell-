# SpyDragon-Windows-Version-PowerShell-
Spy Dragon v.2 es un toolkit de inteligencia OSINT de código abierto diseñado para centralizar en una sola interfaz múltiples capacidades de recolección, análisis y correlación de información pública, facilitando tareas de investigación digital, ciberseguridad y reconocimiento pasivo y activo de objetivos siempre bajo principios de uso ético y con la debida autorización. Este proyecto ofrece un entorno interactivo tipo consola que integra herramientas para análisis de direcciones IP, geolocalización, consultas WHOIS, resolución DNS, inspección de cabeceras HTTP, pruebas de conectividad como ping y traceroute, detección de puertos abiertos y revisión de configuraciones SSL/TLS, así como descubrimiento de subdominios y tecnologías web utilizadas por un sitio, permitiendo obtener una visión técnica completa de infraestructuras en internet. Además, incorpora módulos enfocados en inteligencia de personas y datos como análisis de números telefónicos, identificación de operadoras, geolocalización, análisis de correos electrónicos incluyendo cabeceras y filtraciones en brechas de datos, junto con utilidades de búsqueda avanzada como Google Dorks, consultas en Pastebin y herramientas tipo Sherlock para rastreo de usernames en múltiples plataformas. También incluye funcionalidades orientadas a seguridad y anonimato como detección de VPN o proxies, verificación de nodos de salida TOR, consulta de reputación de IPs y dominios, comparación contra listas negras y análisis de URLs, complementado con capacidades de extracción de metadatos EXIF de imágenes y generación de hashes para integridad de archivos. En el ámbito de redes, Spy Dragon permite obtener información de la red local, escanear dispositivos conectados, analizar puertos específicos y visualizar datos relevantes del entorno, integrando además opciones para obtener la IP pública del usuario y realizar consultas avanzadas como WHOIS de direcciones IP o geolocalización por coordenadas. El sistema está pensado como una herramienta modular, escalable y fácil de usar, ideal tanto para investigadores de seguridad, analistas OSINT y entusiastas de la ciberseguridad que buscan una solución todo en uno para recopilación de inteligencia, documentación y generación de reportes. El repositorio incluye el código fuente listo para su uso y modificación, un README que explica la instalación, dependencias y uso básico del programa, así como un manual que describe cada funcionalidad disponible en el menú, proporcionando ejemplos prácticos y recomendaciones de uso responsable. Spy Dragon promueve el acceso libre al conocimiento y el desarrollo colaborativo, recordando que la información es una herramienta poderosa que debe ser utilizada con responsabilidad, respeto a la privacidad y dentro del marco legal aplicable.
